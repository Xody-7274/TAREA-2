Ejercicio 30: Funciones inversas trigonométricas e hiperbólicas

Nota: Las funciones inversas trigonométricas e hiperbólicas en el plano complejo son multivaluadas debido a la naturaleza multivaluada del logaritmo complejo.

Definiciones usadas:

\[ \sin w = \frac{e^{iw} - e^{-iw}}{2i}, \quad \cos w = \frac{e^{iw} + e^{-iw}}{2}, \quad \tan w = \frac{\sin w}{\cos w} \]
\[ \sinh w = \frac{e^{w} - e^{-w}}{2}, \quad \cosh w = \frac{e^{w} + e^{-w}}{2}, \quad \tanh w = \frac{\sinh w}{\cosh w} \]

Las funciones inversas se definen por las relaciones:

\[ w = \arcsin z \ \text{si} \ \sin w = z, \quad w = \arccos z \ \text{si} \ \cos w = z, \quad \text{etc.} \]

(a) Demostrar: \(\displaystyle \arccos z = -i \ln(z + \sqrt{z^2 - 1})\)

Paso 1: Partir de la definición \(w = \arccos z \Leftrightarrow \cos w = z\).

\[ \cos w = \frac{e^{iw} + e^{-iw}}{2} = z \]

Paso 2: Multiplicar por 2 y reorganizar.

\[ e^{iw} + e^{-iw} = 2z \] Multiplicando por \(e^{iw}\): \[ e^{2iw} + 1 = 2z e^{iw} \] \[ e^{2iw} - 2z e^{iw} + 1 = 0 \]

Paso 3: Resolver la ecuación cuadrática en \(e^{iw}\). Sea \(u = e^{iw}\):

\[ u^2 - 2z u + 1 = 0 \] \[ u = \frac{2z \pm \sqrt{4z^2 - 4}}{2} = z \pm \sqrt{z^2 - 1} \] Por lo tanto, \[ e^{iw} = z \pm \sqrt{z^2 - 1} \]

Paso 4: Aplicar logaritmo y despejar \(w\).

\[ iw = \ln(z \pm \sqrt{z^2 - 1}) \] \[ w = -i \ln(z \pm \sqrt{z^2 - 1}) \]

Dado que el logaritmo complejo es multivaluado, la expresión con el signo \(\pm\) se cubre tomando todas las ramas del logaritmo. Por convención se escribe:

\[ \boxed{\arccos z = -i \ln(z + \sqrt{z^2 - 1})} \]

(b) Demostrar: \(\displaystyle \arcsin z = -i \ln(iz + \sqrt{1 - z^2})\)

Paso 1: Partir de \(w = \arcsin z \Leftrightarrow \sin w = z\).

\[ \sin w = \frac{e^{iw} - e^{-iw}}{2i} = z \]

Paso 2: Multiplicar por \(2i\) y reorganizar.

\[ e^{iw} - e^{-iw} = 2iz \] Multiplicando por \(e^{iw}\): \[ e^{2iw} - 1 = 2iz e^{iw} \] \[ e^{2iw} - 2iz e^{iw} - 1 = 0 \]

Paso 3: Resolver la ecuación cuadrática en \(e^{iw}\). Sea \(u = e^{iw}\):

\[ u^2 - 2iz u - 1 = 0 \] \[ u = \frac{2iz \pm \sqrt{(2iz)^2 + 4}}{2} = \frac{2iz \pm \sqrt{-4z^2 + 4}}{2} = iz \pm \sqrt{1 - z^2} \] Por lo tanto, \[ e^{iw} = iz \pm \sqrt{1 - z^2} \]

Paso 4: Aplicar logaritmo y despejar \(w\).

\[ iw = \ln(iz \pm \sqrt{1 - z^2}) \] \[ w = -i \ln(iz \pm \sqrt{1 - z^2}) \]
\[ \boxed{\arcsin z = -i \ln(iz + \sqrt{1 - z^2})} \]

(c) Demostrar: \(\displaystyle \text{arccosh } z = \ln(z + \sqrt{z^2 - 1})\)

Paso 1: Partir de \(w = \text{arccosh } z \Leftrightarrow \cosh w = z\).

\[ \cosh w = \frac{e^{w} + e^{-w}}{2} = z \]

Paso 2: Proceder de manera análoga a (a), pero con exponenciales reales.

\[ e^{w} + e^{-w} = 2z \] \[ e^{2w} - 2z e^{w} + 1 = 0 \] \[ e^{w} = z \pm \sqrt{z^2 - 1} \] \[ w = \ln(z \pm \sqrt{z^2 - 1}) \]
\[ \boxed{\text{arccosh } z = \ln(z + \sqrt{z^2 - 1})} \]

(d) Demostrar: \(\displaystyle \text{arcsinh } z = \ln(z + \sqrt{z^2 + 1})\)

Paso 1: Partir de \(w = \text{arcsinh } z \Leftrightarrow \sinh w = z\).

\[ \sinh w = \frac{e^{w} - e^{-w}}{2} = z \]

Paso 2: Proceder de manera análoga a (b), pero con exponenciales reales.

\[ e^{w} - e^{-w} = 2z \] \[ e^{2w} - 2z e^{w} - 1 = 0 \] \[ e^{w} = z \pm \sqrt{z^2 + 1} \] \[ w = \ln(z \pm \sqrt{z^2 + 1}) \]
\[ \boxed{\text{arcsinh } z = \ln(z + \sqrt{z^2 + 1})} \]

(e) Demostrar: \(\displaystyle \arctan z = \frac{i}{2} \ln\frac{i + z}{i - z}\)

Paso 1: Partir de \(w = \arctan z \Leftrightarrow \tan w = z\).

\[ \tan w = \frac{\sin w}{\cos w} = \frac{1}{i} \cdot \frac{e^{iw} - e^{-iw}}{e^{iw} + e^{-iw}} = z \]

Paso 2: Simplificar y despejar \(e^{2iw}\).

\[ \frac{e^{iw} - e^{-iw}}{e^{iw} + e^{-iw}} = iz \] Sea \(u = e^{2iw}\), entonces: \[ \frac{u - 1}{u + 1} = iz \] \[ u - 1 = iz(u + 1) \] \[ u - 1 = iz u + iz \] \[ u - iz u = 1 + iz \] \[ u(1 - iz) = 1 + iz \] \[ u = \frac{1 + iz}{1 - iz} \] Multiplicando numerador y denominador por \(i\): \[ u = \frac{i(1 + iz)}{i(1 - iz)} = \frac{i + i^2 z}{i - i^2 z} = \frac{i - z}{i + z} \] Por lo tanto, \[ e^{2iw} = \frac{i - z}{i + z} \quad \Rightarrow \quad e^{-2iw} = \frac{i + z}{i - z} \]

Paso 3: Aplicar logaritmo y despejar \(w\).

\[ -2iw = \ln\left(\frac{i + z}{i - z}\right) \] \[ w = -\frac{1}{2i} \ln\left(\frac{i + z}{i - z}\right) = \frac{i}{2} \ln\left(\frac{i + z}{i - z}\right) \]
\[ \boxed{\arctan z = \frac{i}{2} \ln\frac{i + z}{i - z}} \]

(f) Demostrar: \(\displaystyle \text{arctanh } z = \frac{1}{2} \ln\frac{1 + z}{1 - z}\)

Paso 1: Partir de \(w = \text{arctanh } z \Leftrightarrow \tanh w = z\).

\[ \tanh w = \frac{e^{w} - e^{-w}}{e^{w} + e^{-w}} = z \]

Paso 2: Despejar \(e^{2w}\).

\[ e^{w} - e^{-w} = z(e^{w} + e^{-w}) \] \[ e^{w} - e^{-w} = z e^{w} + z e^{-w} \] \[ e^{w} - z e^{w} = e^{-w} + z e^{-w} \] \[ e^{w}(1 - z) = e^{-w}(1 + z) \] \[ e^{2w} = \frac{1 + z}{1 - z} \]

Paso 3: Aplicar logaritmo y despejar \(w\).

\[ 2w = \ln\left(\frac{1 + z}{1 - z}\right) \] \[ w = \frac{1}{2} \ln\left(\frac{1 + z}{1 - z}\right) \]
\[ \boxed{\text{arctanh } z = \frac{1}{2} \ln\frac{1 + z}{1 - z}} \]

(g) Multivaluación de \(w = \arcsin z\)

Paso 1: Recordar la expresión de \(\arcsin z\) obtenida en (b):

\[ \arcsin z = -i \ln(iz + \sqrt{1 - z^2}) \]

El logaritmo complejo es infinitamente multivaluado:

\[ \ln u = \ln|u| + i \arg u + 2k\pi i, \quad k \in \mathbb{Z} \]

Paso 2: Si \(w_1\) es un valor particular de \(\arcsin z\) (correspondiente a una rama del logaritmo y una rama de la raíz cuadrada), entonces cualquier otro valor difiere por:

\[ w = w_1 + 2n\pi, \quad n \in \mathbb{Z} \]

debido a que \(-i \cdot 2k\pi i = 2k\pi\).

Paso 3: Además, la raíz cuadrada introduce otra rama. Consideremos el signo opuesto en la expresión \(iz \pm \sqrt{1 - z^2}\).

Si \(w_1\) corresponde al signo \(+\) (es decir, \(e^{iw_1} = iz + \sqrt{1 - z^2}\)), y sea \(w'\) tal que \(e^{iw'} = iz - \sqrt{1 - z^2}\). Entonces:

\[ iz - \sqrt{1 - z^2} = \frac{-1}{iz + \sqrt{1 - z^2}} = - e^{-iw_1} \] \[ e^{iw'} = - e^{-iw_1} = e^{i\pi} e^{-iw_1} = e^{i(\pi - w_1)} \] Por lo tanto, \[ w' = \pi - w_1 + 2n\pi, \quad n \in \mathbb{Z} \]

Paso 4: Combinando ambas, todos los valores de \(\arcsin z\) son de la forma:

\[ \boxed{w = w_1 + 2n\pi \quad \text{y} \quad w = \pi - w_1 + 2n\pi, \quad n = 0, \pm 1, \pm 2, \dots} \]

Esto muestra que \(\arcsin z\) es infinitamente multivaluado.

Valor principal de \(\arcsin z\)

El valor principal de \(w = u + iv = \arcsin z\) se define como aquel que satisface:

\[ -\frac{\pi}{2} \leq u \leq \frac{\pi}{2} \ \text{si} \ v \geq 0, \quad -\frac{\pi}{2} < u < \frac{\pi}{2} \ \text{si} \ v < 0 \]

Esto selecciona una rama particular entre las infinitas posibilidades.

Resumen

Todas las funciones inversas trigonométricas e hiperbólicas en el plano complejo pueden expresarse en términos del logaritmo complejo, lo que las hace inherentemente multivaluadas. Las fórmulas demostradas son fundamentales para el análisis complejo y tienen aplicaciones en la integración de funciones complejas y en la resolución de ecuaciones.
