Complex Numbers and Functions. Complex Differentiation

Equations with Complex Logarithms

Problem Set 13.7 - Autor: Martínez León Josué David

Ejercicio 18: Resolver para z

Resolver la siguiente ecuación para z:

\[\ln z = - \frac{\pi i}{2}\]

Solución:

Paso 1: Recordar la definición del logaritmo complejo y su relación inversa con la exponencial

El logaritmo complejo se define como:

\[ \ln z = \ln|z| + i(\arg z + 2k\pi), \quad k \in \mathbb{Z} \]

La relación inversa es:

\[ \text{Si } \ln z = w, \text{ entonces } z = e^{w} \]

Esta relación es válida porque la función exponencial es la inversa del logaritmo (aunque el logaritmo es multivaluado, la exponencial no lo es).

Paso 2: Aplicar la función exponencial a ambos lados de la ecuación

Dado que \(\ln z = -\frac{\pi i}{2}\), aplicamos la exponencial:

\[ z = e^{-\frac{\pi i}{2}} \]

Paso 3: Evaluar la exponencial compleja

Usamos la fórmula de Euler: \(e^{i\theta} = \cos\theta + i\sin\theta\)

\[ z = e^{-\frac{\pi i}{2}} = \cos\left(-\frac{\pi}{2}\right) + i\sin\left(-\frac{\pi}{2}\right) \]
\[ = \cos\left(-\frac{\pi}{2}\right) + i\sin\left(-\frac{\pi}{2}\right) \]
\[ = 0 + i(-1) = -i \]

Por lo tanto:

\[ z = -i \]

Paso 4: Verificación

Verificamos que \(\ln(-i) = -\frac{\pi i}{2}\) (al menos uno de sus valores):

\[ -i = 0 - 1i \]

Módulo: \(|z| = \sqrt{0^2 + (-1)^2} = 1\)

Argumento: \(\arg(-i) = -\frac{\pi}{2}\) (o equivalentemente, \(\frac{3\pi}{2}\))

Aplicando la definición del logaritmo complejo:

\[ \ln(-i) = \ln(1) + i\left(-\frac{\pi}{2} + 2k\pi\right) = 0 + i\left(-\frac{\pi}{2} + 2k\pi\right) \]

Para \(k = 0\), obtenemos:

\[ \ln(-i) = -\frac{\pi i}{2} \]

¡La verificación es exitosa!

Nota importante: La ecuación \(\ln z = -\frac{\pi i}{2}\) significa que \(-\frac{\pi i}{2}\) es uno de los valores del logaritmo complejo de \(z\). Aunque el logaritmo complejo es multivaluado, la función exponencial nos da una única solución \(z = e^{-\frac{\pi i}{2}} = -i\), ya que diferentes valores del logaritmo corresponden a diferentes ramas, pero la exponencial es univaluada.
Resultado Final:
\[ \boxed{z = -i} \]
\[ \text{Forma rectangular: } z = 0 - i \]
\[ \text{Módulo: } |z| = 1 \]
\[ \text{Argumento principal: } \arg z = -\frac{\pi}{2} \text{ radianes } = -90° \]

Interpretación Geométrica:

La solución \(z = -i\) se encuentra sobre el círculo unitario en el plano complejo, específicamente en el punto (0, -1). Esto corresponde a un número complejo con:

  • Módulo 1: Distancia desde el origen hasta el punto es 1
  • Argumento -π/2: Forma un ángulo de -90° con el eje real positivo

Cuando calculamos el logaritmo complejo de \(-i\), obtenemos:

\[ \ln(-i) = \ln(1) + i\left(-\frac{\pi}{2} + 2k\pi\right) = i\left(-\frac{\pi}{2} + 2k\pi\right) \]

Para \(k = 0\), obtenemos exactamente \(-\frac{\pi i}{2}\), que es el valor dado en la ecuación original.

Esta solución ilustra que la ecuación \(\ln z = w\) tiene una única solución \(z = e^w\), a pesar de que el logaritmo complejo es multivaluado.
